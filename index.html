Re-Learning Git and GitHub

Common Commands:
  • Clone: This is used to make a copy of your repository on Git Hub available on your local machine.
  • Add: When new changes are made, deleted or new files are made, you use this command to have git track those changes.
  • Commit: Saves your changes
  • Push: This will upload your changes to a remote repository on GitHub
  • Pull: When changes have been made to the remote repository you want to use the “pull” command to download those changes.
  • Status: Shares what changes have been made but are not being tracked
